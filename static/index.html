Pixar Movie API

This API provides information about Pixar movies.
API DOCS: https://pixar-api.vercel.app/docs

Get All Films

Retrieve a list of all Pixar films.

GET https://pixar-api.vercel.app/api/films/

Get Film by ID

Retrieve details of a specific film using its ID.

GET https://pixar-api.vercel.app/api/films/{move name}

Example: https://pixar-api.vercel.app/api/films/Cars
